# Bashly AI Prompts

- prompt.txt - is for the custom GPT chat at [bashly.dev/chat](https://bashly.dev/chat)
- AGENTS.md - is a file that can be used with local coding agents

## From zero to bashly script in 5 minutes

- Place AGENTS.md in a new empty directory
- Run something like this:
  ```shell
  codex exec "build a bashly-based bash script for me that simply outputs inspirational and motivational funny quotes and phrases. I want teh command line to have subcommands for different categories, tech oriented (like devops, ruby, cloud) and each will print nice funny random single message"
  ```
